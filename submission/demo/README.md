# Demo Video

Save your Stage 2 screen recording here or upload it and keep the link in your submission form.

## Recording checklist

1. Start backend (`notification_app_be` on port 4000)
2. Start frontend (`notification_app_fe` on port 3000)
3. Show desktop view:
   - All notifications page
   - Change type filter
   - Mark one notification as viewed
   - Open priority inbox page
4. Switch to mobile view and repeat the same flow
5. Keep the recording under 5 minutes

`campus-notifications-demo.mp4`
